
AngularJs- is client side processing language. it's library of javascript
           introduced by Google.com
		   
->Angularjs is MVC framework
    
	M ->MODULES
	V ->VIEWS
	C ->CONTROLLERS
	
	->VIEWS : it manage the display part(user interface html,css,bootstrap)
	->MODULES : it manage the processing part (using js code input will be 
	     	 processed)
	
    ->CONTROLLER : it Control the flow of data from views to modules
